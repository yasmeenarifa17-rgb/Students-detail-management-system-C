# 🎓 Student Management System (C Language)

## 📌 Project Overview
This project is a simple and efficient Student Management System developed using C programming language. It allows users to manage student records easily using file handling.

## ✨ Features
- ➕ Add Student Details
- 📋 Display All Students
- 🔍 Search Student by Roll Number
- ✏️ Update Student Details
- ❌ Delete Student Record
- 📊 Count Total Students

## 🛠️ Technologies Used
- C Programming Language
- File Handling (Binary Files)
